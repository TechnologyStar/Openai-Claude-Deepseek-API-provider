## 📖 使用指南

1. 🔑 **获取 API Key**：注册平台后，在用户中心复制您的API密钥
2. ⚙ **配置 Endpoint**：填写 API 地址到支持的应用程序中
3. 🤖 **选择模型**：根据平台支持情况切换不同的AI模型
4. 📊 **用量监控**：推荐使用客户端自带的用量限制功能

### 详细配置步骤

#### 步骤一：选择合适的API提供商
- 根据您的需求选择免费或付费服务
- 查看标签了解平台支持的模型类型
- 注意是否需要国际网络访问

#### 步骤二：注册并获取API Key
- 访问选定的平台并完成注册
- 在用户中心或API设置页面找到您的API密钥
- 妥善保管API密钥，不要泄露给他人

#### 步骤三：配置应用程序
- 在您使用的AI应用中找到API设置
- 填入API端点地址和您的API密钥
- 选择合适的模型进行测试
