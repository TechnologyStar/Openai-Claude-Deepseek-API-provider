### 一、Cherry Studio

> [Cherry Studio](https://github.com/CherryHQ/cherry-studio) 是一款跨平台的多模型AI桌面客户端，支持 Windows、macOS、Linux 以及 iOS 和 Android 移动端。
>
> **主要功能：**
>
> * **多模型支持**：集成 OpenAI、Anthropic、Gemini、Claude、Perplexity 等主流云服务，支持本地模型如 Ollama、LM Studio。
> * **智能助手系统**：内置300+预设助手，支持自定义角色与多模型并行对话。
> * **文档与数据处理**：支持文本、图片、Office、PDF 等多种格式，提供 WebDAV 文件管理与备份。
> * **实用工具集成**：包括全局搜索、主题管理、AI 翻译、拖拽排序、小程序支持等。
> * **增强的用户体验**：提供完整的 Markdown 渲染、浅色/深色主题、透明窗口、内容分享等功能。
>
> 适用于开发者、内容创作者、研究人员等多种用户群体，提供高效的AI交互体验。
>
> [**查看应用**](https://github.com/CherryHQ/cherry-studio)([GitHub][1])

### 二、ChatGPT.好友插件

> 该插件是 uTools 平台上的一款强大 AI 聊天工具，支持创建多个 AI 好友，每个好友可独立设置角色、指令、模型参数和对话记录。
>
> **主要功能：**
>
> * 支持上下文连续对话，可开启或关闭。
> * 每个 AI 好友可独立设置角色、指令、调参、模型、对话记录等。
> * 支持 DALL·E 3 模型绘画、图生图、语音录制与播报。
> * 支持最新的 OpenAI 模型，包括 GPT-4。
> * 支持最新的助理 API：可创建助理、管理线程、上传文件分析。
> * API Key 和对话记录均加密存储，保障数据安全。
>
> 该插件适用于桌面端，暂不支持移动端。
>
> [**查看应用**](https://u.tools/plugins/detail/ChatGPT.%E5%A5%BD%E5%8F%8B/)

> **主要功能：**

> * 一键部署，支持 Vercel 等平台，方便快捷。

## 使用指南
1. 获取API Key：注册对应平台后，在用户中心获取

3. 模型选择：不同平台支持的模型列表可能不同
4. 用量监控：建议在客户端设置用量限制
