## 📖 使用指南

1. 🔑 获取 API Key：注册平台后，在用户中心复制
2. ⚙ 配置 Endpoint：填写 API 地址到支持应用中
3. 🤖 选择模型：根据平台支持情况切换不同模型
4. 📊 用量监控：推荐使用客户端自带用量限制功能
